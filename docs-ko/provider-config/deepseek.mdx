Cline은 `deepseek-chat` 및 `deepseek-reasoner`를 포함하여 DeepSeek API를 통해 모델에 액세스하는 것을 지원합니다.

**웹사이트:** [https://platform.deepseek.com/](https://platform.deepseek.com/)

### API 키 받기

1.  **가입/로그인:** [DeepSeek 플랫폼](https://platform.deepseek.com/)으로 이동합니다. 계정을 만들거나 로그인합니다.
2.  **API 키로 이동:** 플랫폼의 [API 키](https://platform.deepseek.com/api_keys) 섹션에서 API 키를 찾습니다.
3.  **키 만들기:** "새 API 키 만들기"를 클릭합니다. 키에 설명적인 이름(예: "Cline")을 지정합니다.
4.  **키 복사:** **중요:** API 키를 _즉시_ 복사합니다. 다시 볼 수 없습니다. 안전하게 보관하십시오.

### 지원되는 모델

Cline은 다음 DeepSeek 모델을 지원합니다.

-   `deepseek-v3-0324` (코딩 작업에 권장)
-   `deepseek-r1` (추론 작업에 권장)

### Cline의 구성

1.  **Cline 설정 열기:** Cline 패널에서 ⚙️ 아이콘을 클릭합니다.
2.  **공급자 선택:** "API 공급자" 드롭다운에서 "DeepSeek"을 선택합니다.
3.  **API 키 입력:** DeepSeek API 키를 "DeepSeek API 키" 필드에 붙여넣습니다.
4.  **모델 선택:** "모델" 드롭다운에서 원하는 모델을 선택합니다.

### 팁 및 참고 사항

-   **가격 책정:** 모델 비용에 대한 자세한 내용은 [DeepSeek 가격 책정](https://api-docs.deepseek.com/quick_start/pricing/) 페이지를 참조하십시오.
