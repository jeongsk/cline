Cline은 공식 OpenAI API를 통해 직접 모델에 액세스하는 것을 지원합니다.

**웹사이트:** [https://openai.com/](https://openai.com/)

### API 키 받기

1.  **가입/로그인:** [OpenAI 플랫폼](https://platform.openai.com/)을 방문합니다. 계정을 만들거나 이미 계정이 있는 경우 로그인해야 합니다.
2.  **API 키로 이동:** 로그인한 후 계정의 [API 키 섹션](https://platform.openai.com/api-keys)으로 이동합니다.
3.  **키 만들기:** "새 비밀 키 만들기"를 클릭합니다. 키에 설명적인 이름(예: "Cline API 키")을 지정하는 것이 좋습니다.
4.  **키 복사:** **중요:** 생성된 API 키를 즉시 복사합니다. 보안상의 이유로 OpenAI는 다시 표시하지 않습니다. 이 키를 안전하고 보안된 위치에 보관하십시오.

### 지원되는 모델

Cline은 다음을 포함하되 이에 국한되지 않는 다양한 OpenAI 모델과 호환됩니다.

-   'o3'
-   `o3-mini` (중간 추론 노력)
-   'o4-mini'
-   `o3-mini-high` (높은 추론 노력)
-   `o3-mini-low` (낮은 추론 노력)
-   `o1`
-   `o1-preview`
-   `o1-mini`
-   `gpt-4.5-preview`
-   `gpt-4o`
-   `gpt-4o-mini`
-   'gpt-4.1'
-   'gpt-4.1-mini'

사용 가능한 모델 및 해당 기능의 최신 목록은 공식 [OpenAI 모델 설명서](https://platform.openai.com/docs/models)를 참조하십시오.

### Cline의 구성

1.  **Cline 설정 열기:** Cline 패널에서 설정 기어 아이콘(⚙️)을 클릭합니다.
2.  **공급자 선택:** "API 공급자" 드롭다운 메뉴에서 "OpenAI"를 선택합니다.
3.  **API 키 입력:** OpenAI API 키를 "OpenAI API 키" 필드에 붙여넣습니다.
4.  **모델 선택:** "모델" 드롭다운 목록에서 원하는 모델을 선택합니다.
5.  **(선택 사항) 기본 URL:** OpenAI API에 프록시 또는 사용자 지정 기본 URL을 사용해야 하는 경우 여기에 입력할 수 있습니다. 대부분의 사용자는 기본값에서 이 설정을 변경할 필요가 없습니다.

### 팁 및 참고 사항

-   **가격 책정:** 다양한 모델과 관련된 비용에 대한 자세한 내용은 [OpenAI 가격 책정 페이지](https://openai.com/pricing)를 검토하십시오.
-   **Azure OpenAI 서비스:** Azure OpenAI 서비스를 사용하려는 경우 Cline과 함께 Azure OpenAI에 대한 특정 설명서를 별도로 찾거나 Cline이 사용자 지정 구성을 위해 해당 기능을 지원하는 경우 OpenAI 호환 엔드포인트로 구성해야 할 수 있습니다.
