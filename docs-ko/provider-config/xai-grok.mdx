xAI는 대화 능력과 큰 컨텍스트 창으로 알려진 대규모 언어 모델인 Grok을 개발한 회사입니다. Grok 모델은 유용하고 유익하며 상황에 맞는 응답을 제공하도록 설계되었습니다.

**웹사이트:** [https://x.ai/](https://x.ai/)

### API 키 받기

1.  **가입/로그인:** [xAI 콘솔](https://console.x.ai/)로 이동합니다. 계정을 만들거나 로그인합니다.
2.  **API 키로 이동:** 대시보드의 API 키 섹션으로 이동합니다.
3.  **키 만들기:** 새 API 키를 만들려면 클릭합니다. 키에 설명적인 이름(예: "Cline")을 지정합니다.
4.  **키 복사:** **중요:** API 키를 _즉시_ 복사합니다. 다시 볼 수 없습니다. 안전하게 보관하십시오.

### 지원되는 모델

Cline은 다음 xAI Grok 모델을 지원합니다.

#### Grok-3 모델

-   `grok-3-beta` (기본값) - 131K 컨텍스트 창이 있는 xAI의 Grok-3 베타 모델
-   `grok-3-fast-beta` - 131K 컨텍스트 창이 있는 xAI의 Grok-3 빠른 베타 모델
-   `grok-3-mini-beta` - 131K 컨텍스트 창이 있는 xAI의 Grok-3 미니 베타 모델
-   `grok-3-mini-fast-beta` - 131K 컨텍스트 창이 있는 xAI의 Grok-3 미니 빠른 베타 모델

#### Grok-2 모델

-   `grok-2-latest` - xAI의 Grok-2 모델 - 131K 컨텍스트 창이 있는 최신 버전
-   `grok-2` - 131K 컨텍스트 창이 있는 xAI의 Grok-2 모델
-   `grok-2-1212` - 131K 컨텍스트 창이 있는 xAI의 Grok-2 모델 (버전 1212)

#### Grok 비전 모델

-   `grok-2-vision-latest` - xAI의 Grok-2 비전 모델 - 이미지 지원 및 32K 컨텍스트 창이 있는 최신 버전
-   `grok-2-vision` - 이미지 지원 및 32K 컨텍스트 창이 있는 xAI의 Grok-2 비전 모델
-   `grok-2-vision-1212` - 이미지 지원 및 32K 컨텍스트 창이 있는 xAI의 Grok-2 비전 모델 (버전 1212)
-   `grok-vision-beta` - 이미지 지원 및 8K 컨텍스트 창이 있는 xAI의 Grok 비전 베타 모델

#### 레거시 모델

-   `grok-beta` - 131K 컨텍스트 창이 있는 xAI의 Grok 베타 모델 (레거시)

### Cline의 구성

1.  **Cline 설정 열기:** Cline 패널에서 설정 아이콘(⚙️)을 클릭합니다.
2.  **공급자 선택:** "API 공급자" 드롭다운에서 "xAI"를 선택합니다.
3.  **API 키 입력:** xAI API 키를 "xAI API 키" 필드에 붙여넣습니다.
4.  **모델 선택:** "모델" 드롭다운에서 원하는 Grok 모델을 선택합니다.

### 추론 기능

Grok 3 Mini 모델은 특수 추론 기능을 갖추고 있어 복잡한 문제 해결 작업에 특히 유용한 "응답하기 전에 생각"할 수 있습니다.

#### 추론 지원 모델

추론은 다음에 의해서만 지원됩니다.

-   `grok-3-mini-beta`
-   `grok-3-mini-fast-beta`

Grok 3 모델 `grok-3-beta` 및 `grok-3-fast-beta`는 추론을 지원하지 않습니다.

#### 추론 노력 제어

추론 지원 모델을 사용하는 경우 `reasoning_effort` 매개변수를 사용하여 모델이 얼마나 열심히 생각하는지 제어할 수 있습니다.

-   `low`: 최소한의 생각 시간, 빠른 응답을 위해 더 적은 토큰 사용
-   `high`: 최대 생각 시간, 복잡한 문제에 더 많은 토큰 활용

빨리 완료되어야 하는 간단한 쿼리에는 `low`를 선택하고 응답 대기 시간이 덜 중요한 어려운 문제에는 `high`를 선택합니다.

#### 주요 기능

-   **단계별 문제 해결**: 모델은 답변을 제공하기 전에 문제를 체계적으로 생각합니다.
-   **수학 및 정량적 강점**: 수치 문제 및 논리 퍼즐에 탁월합니다.
-   **추론 추적 액세스**: 모델의 사고 과정은 응답 완료 개체의 `reasoning_content` 필드를 통해 사용할 수 있습니다.

### 팁 및 참고 사항

-   **컨텍스트 창:** 대부분의 Grok 모델은 큰 컨텍스트 창(최대 131K 토큰)을 특징으로 하여 프롬프트에 상당한 양의 코드와 컨텍스트를 포함할 수 있도록 합니다.
-   **비전 기능:** 이미지를 처리하거나 분석해야 하는 경우 비전 지원 모델(`grok-2-vision-latest`, `grok-2-vision` 등)을 선택합니다.
-   **가격 책정:** 가격은 모델에 따라 다르며 입력 비용은 백만 토큰당 0.3달러에서 5.0달러, 출력 비용은 백만 토큰당 0.5달러에서 25.0달러입니다. 가장 최신 가격 정보는 xAI 설명서를 참조하십시오.
-   **성능 절충:** "빠른" 변형은 일반적으로 더 빠른 응답 시간을 제공하지만 비용이 더 높을 수 있으며 "미니" 변형은 더 경제적이지만 기능이 저하될 수 있습니다.
