Cline은 [Requesty](https://www.requesty.ai/) AI 플랫폼을 통해 모델에 액세스하는 것을 지원합니다. Requesty는 150개 이상의 대규모 언어 모델(LLM)과 상호 작용하기 위한 쉽고 최적화된 API를 제공합니다.

**웹사이트:** [https://www.requesty.ai/](https://www.requesty.ai/)

### API 키 받기

1.  **가입/로그인:** [Requesty 웹사이트](https://www.requesty.ai/)로 이동하여 계정을 만들거나 로그인합니다.
2.  **API 키 받기:** Requesty 대시보드의 [API 관리](https://app.requesty.ai/manage-api) 섹션에서 API 키를 받을 수 있습니다.

### 지원되는 모델

Requesty는 광범위한 모델에 대한 액세스를 제공합니다. Cline은 사용 가능한 모델의 최신 목록을 자동으로 가져옵니다. 사용 가능한 모델의 전체 목록은 [모델 목록](https://app.requesty.ai/router/list) 페이지에서 볼 수 있습니다.

### Cline의 구성

1.  **Cline 설정 열기:** Cline 패널에서 설정 아이콘(⚙️)을 클릭합니다.
2.  **공급자 선택:** "API 공급자" 드롭다운에서 "Requesty"를 선택합니다.
3.  **API 키 입력:** Requesty API 키를 "Requesty API 키" 필드에 붙여넣습니다.
4.  **모델 선택:** "모델" 드롭다운에서 원하는 모델을 선택합니다.

### 팁 및 참고 사항

-   **최적화**: Requesty는 비용을 낮추기 위한 다양한 실시간 비용 최적화를 제공합니다.
-   **통합되고 간소화된 청구**: 단일 [API 키](https://app.requesty.ai/manage-api)를 통해 모든 공급자 및 모델에 대한 무제한 액세스, 자동 잔액 충전 등을 제공합니다.
-   **비용 추적**: [비용 대시보드](https://app.requesty.ai/cost-management) 또는 [Requesty VS Code 확장 프로그램](https://marketplace.visualstudio.com/items?itemName=Requesty.requesty)을 통해 모델별, 코딩 언어별, 변경된 파일별 비용 등을 추적합니다.
-   **통계 및 로그**: [코딩 통계 대시보드](https://app.requesty.ai/usage-stats)를 확인하거나 [LLM 상호 작용 로그](https://app.requesty.ai/logs)를 검토합니다.
-   **대체 정책**: 공급자가 다운되었을 때 대체 정책을 통해 LLM이 계속 작동하도록 합니다.
-   **프롬프트 캐싱:** 일부 공급자는 프롬프트 캐싱을 지원합니다. [캐싱 기능이 있는 모델 검색](https://app.requesty.ai/router/list).
