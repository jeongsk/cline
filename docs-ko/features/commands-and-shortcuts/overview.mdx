Cline은 VSCode 인터페이스에 직접 통합되어 워크플로를 중단하지 않고 AI 지원에 액세스할 수 있도록 합니다. 이러한 통합은 편집기 전체에서 컨텍스트 메뉴, 키보드 단축키 및 빠른 수정 사항의 명령어로 나타납니다.

### 편집기 통합이란 무엇인가요?

편집기 통합은 작업 중인 곳에서 바로 Cline을 사용할 수 있게 해주는 명령어와 단축키입니다. 먼저 Cline 패널로 전환하는 대신 코드를 선택하고 마우스 오른쪽 버튼을 클릭한 다음 즉시 Cline으로 보내 도움을 받을 수 있습니다.
이러한 통합은 VSCode 전체의 다양한 위치에 나타납니다.

-   편집기 컨텍스트 메뉴(마우스 오른쪽 버튼 클릭 메뉴) - "Cline에 추가"
-   터미널 컨텍스트 메뉴 - "Cline에 추가"
-   소스 제어 보기 - "커밋 메시지 생성"
-   키보드 단축키 - 다양한 Cline 명령어
-   빠른 수정 옵션(전구 메뉴) - "Cline으로 수정", "Cline으로 설명", "Cline으로 개선"

### 작동 방식

이러한 명령어를 사용하면 Cline은 다음을 수행합니다.

-   관련 컨텍스트(선택한 코드, 파일 경로, 터미널 출력 등)를 캡처합니다.
-   Cline 인터페이스에 포커스를 맞춥니다.
-   캡처된 컨텍스트로 대화를 만듭니다.
-   경우에 따라 제안된 프롬프트를 자동으로 생성합니다.

내부적으로 이러한 명령어는 VSCode의 확장 API를 사용하여 명령어를 등록하고, 편집기 상태에 액세스하고, VSCode 인터페이스를 제어합니다.
