Cline으로 파일을 드래그 앤 드롭하는 것은 이미지, 코드 및 기타 파일을 대화에 빠르게 추가하는 방법입니다.

<Note>VS Code의 특성으로 인해 Cline 채팅 입력으로 파일을 드래그 앤 드롭하려면 드래그하는 동안 `Shift` 키를 누르고 있어야 합니다.</Note>

작업 공간 파일을 Cline으로 드래그 앤 드롭하면 자동으로 [파일 언급](/features/at-mentions/file-mentions)이 생성됩니다. 이렇게 하면 경로를 입력할 필요 없이 대화에서 파일을 참조할 수 있습니다.

### 지원되는 파일 형식

Cline은 파일 시스템의 외부 이미지는 물론 작업 공간의 파일을 드래그하는 것을 지원합니다.
