### 설치 옵션

-   **VS Code 마켓플레이스(권장):** 표준 VS Code 및 Cursor 사용자에게 가장 빠른 방법입니다.
-   **Open VSX 레지스트리:** VSCodium과 같은 VS Code 호환 편집기용입니다.

### 🛠️ VS Code 마켓플레이스: 단계별 설정

Cline을 시작하고 실행하려면 다음 단계를 따르십시오.

1.  **VS Code 열기:** VS Code 애플리케이션을 시작합니다.

    > ⚠️ **참고:** VS Code에 "확장 프로그램 실행 중..."이 표시되면 "허용"을 클릭합니다.

2.  **Cline 폴더 열기:** VS Code에서 문서에 만든 Cline 폴더를 엽니다.
3.  **확장 프로그램으로 이동:** VS Code 측면의 활동 표시줄에서 확장 프로그램 아이콘을 클릭합니다(`Ctrl + Shift + X` 또는 `Cmd + Shift + X`).
4.  **'Cline' 검색:** 확장 프로그램 검색 창에 `Cline`을 입력합니다.

1. **확장 프로그램 설치:** Cline 확장 프로그램 옆에 있는 "설치" 버튼을 클릭합니다.
2. **Cline 열기:**
    - 활동 표시줄에서 Cline 아이콘을 클릭합니다.
    - 또는 명령 팔레트(`Ctrl/Cmd + Shift + P`)를 사용하고 더 나은 보기를 위해 "Cline: 새 탭에서 열기"를 입력합니다.
3. **문제 해결:** Cline 아이콘이 보이지 않으면 VS Code를 다시 시작해 보십시오.

> ✅ **전문가 팁:** VS Code 편집기에 Cline 채팅 창이 나타나야 합니다!

### 🌐 Open VSX 레지스트리

마켓플레이스 액세스 권한이 없는 VS Code 호환 편집기(예: VSCodium 및 Windsurf)의 경우:

1. 편집기를 엽니다.
2. 확장 프로그램 보기에 액세스합니다.
3. "Cline"을 검색합니다.
4. saoudrizwan의 "Cline"을 선택하고 **설치**를 클릭합니다.
5. 메시지가 표시되면 다시 로드합니다.

### 👤 Cline 계정 만들기

이제 Cline이 설치되었으므로 계정 설정을 시작하겠습니다.

1. **Cline에 로그인:**
    - Cline 확장 프로그램에서 **로그인** 버튼을 클릭합니다.
    - 계정을 만들려면 [app.cline.bot](https://app.cline.bot)으로 이동합니다.
2. **무료 크레딧으로 시작:**
    - 신용 카드가 필요하지 않습니다!
3. **사용 가능한 AI 모델:**
    - Anthropic Claude 3.5-Sonnet(코딩 권장)
    - DeepSeek Chat(비용 효율적인 대안)
    - Google Gemini 2.0 Flash
    - 그리고 더 많은 모델 — 모두 Cline 계정을 통해 사용할 수 있습니다.

### 💻 Cline과의 첫 번째 상호 작용

이제 빌드를 시작할 준비가 되었습니다! 이 프롬프트를 복사하여 Cline 채팅 창에 붙여넣습니다.

```
안녕하세요 Cline! Cline 디렉터리에 "hello-world"라는 새 프로젝트 폴더를 만들고 큰 파란색 텍스트로 "Hello World"라고 표시되는 간단한 웹페이지를 만드는 데 도움을 주시겠어요?
```

> ✅ **전문가 팁:** Cline이 프로젝트 폴더를 만들고 첫 번째 웹페이지를 설정하는 데 도움을 줄 것입니다!
