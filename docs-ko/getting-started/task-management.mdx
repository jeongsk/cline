# 작업 관리

Cline을 사용하면 시간이 지남에 따라 많은 작업이 누적됩니다. 작업 관리 시스템은 작업 공간을 효율적으로 유지하기 위해 작업 기록을 구성, 필터링, 검색 및 정리하는 데 도움이 됩니다.

## 작업 기록 액세스

다음과 같은 방법으로 작업 기록에 액세스할 수 있습니다.

1. Cline 사이드바에서 "기록" 버튼 클릭
2. 명령 팔레트를 사용하여 "Cline: 작업 기록 표시" 검색

## 작업 기록 기능

작업 기록 보기에는 다음과 같은 몇 가지 강력한 기능이 제공됩니다.

### 검색 및 필터링

-   **검색 창**: 상단의 퍼지 검색을 사용하여 내용으로 작업을 빠르게 찾습니다.
-   **정렬 옵션**: 다음 기준으로 작업 정렬:
    -   최신순(기본값)
    -   오래된순
    -   가장 비싼 순(API 비용이 가장 높은 순)
    -   가장 많은 토큰 순(토큰 사용량이 가장 높은 순)
    -   가장 관련성 높은 순(검색 시)
-   **즐겨찾기 필터**: 즐겨찾기한 작업만 표시하도록 전환합니다.

### 작업 작업

기록 보기의 각 작업에는 다음과 같은 몇 가지 작업을 사용할 수 있습니다.

-   **열기**: 작업을 클릭하여 Cline 채팅에서 다시 엽니다.
-   **즐겨찾기**: 별표 아이콘을 클릭하여 작업을 즐겨찾기로 표시합니다.
-   **삭제**: 개별 작업을 제거합니다(즐겨찾기는 삭제로부터 보호됨).
-   **내보내기**: 작업 대화를 마크다운으로 내보냅니다.

## ⭐ 작업 즐겨찾기

즐겨찾기 기능을 사용하면 보존하고 빠르게 찾고 싶은 중요한 작업을 표시할 수 있습니다.

### 즐겨찾기 작동 방식

-   **즐겨찾기 표시**: 작업 옆에 있는 별표 아이콘을 클릭하여 즐겨찾기 상태를 전환합니다.
-   **보호**: 즐겨찾기한 작업은 개별 및 일괄 삭제 작업으로부터 보호됩니다(재정의 가능).
-   **필터링**: 즐겨찾기 필터를 사용하여 중요한 작업에 빠르게 액세스합니다.

## 일괄 작업

작업 기록 보기에서는 다음과 같은 몇 가지 일괄 작업을 지원합니다.

-   **여러 항목 선택**: 확인란을 사용하여 여러 작업을 선택합니다.
-   **모두 선택/선택 안 함**: 모든 작업을 빠르게 선택하거나 선택 취소합니다.
-   **선택한 항목 삭제**: 선택한 모든 작업을 제거합니다.
-   **모두 삭제**: 기록에서 모든 작업을 제거합니다(포함하도록 선택하지 않는 한 즐겨찾기는 유지됨).

## 모범 사례

1. **중요한 작업 즐겨찾기**: 참조 작업이나 자주 액세스하는 대화를 즐겨찾기로 표시합니다.
2. **정기적인 정리**: 성능 향상을 위해 오래되거나 사용하지 않는 작업을 주기적으로 제거합니다.
3. **검색 사용**: 퍼지 검색을 활용하여 특정 대화를 빠르게 찾습니다.
4. **가치 있는 작업 내보내기**: 외부 참조를 위해 중요한 작업을 마크다운으로 내보냅니다.

작업 관리는 Cline을 사용할 때 체계적인 워크플로를 유지하는 데 도움이 되므로 과거 대화를 빠르게 찾고, 중요한 작업을 보존하고, 기록을 깨끗하고 효율적으로 유지할 수 있습니다.
