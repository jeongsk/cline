> 💡 **팁:** 코딩이 처음이라면 각 단계를 천천히 진행하세요. 서두를 필요 없습니다. Cline이 안내해 드릴 것입니다!

### 🚀 시작하기

코딩을 시작하기 전에 다음 필수 사항이 준비되었는지 확인하십시오.

#### 1. **VS Code**

인기 있고 무료이며 강력한 코드 편집기입니다.

-   [<u>VS Code 다운로드</u>](https://code.visualstudio.com/)

📺 **추천 유튜브 튜토리얼:** [<u>VS Code 설치 방법</u>](https://www.youtube.com/watch?v=MlIzFUI1QGA)

> ✅ **전문가 팁:** 도크 또는 시작 메뉴에서 쉽게 액세스할 수 있도록 응용 프로그램 폴더(macOS) 또는 Program Files(Windows)에 VS Code를 설치하십시오.

#### 2. **프로젝트 정리**

모든 코딩 프로젝트를 위해 문서 폴더에 `Cline`이라는 전용 폴더를 만듭니다.

-   **macOS:** `/Users/[사용자 이름]/Documents/Cline`
-   **Windows:** `C:\Users\[사용자 이름]\Documents\Cline`

`Cline` 폴더 내에서 프로젝트를 명확하게 구성합니다.

-   `Documents/Cline/workout-app` _(예: 피트니스 추적 앱)_
-   `Documents/Cline/portfolio-website` _(예: 작업물을 보여주기 위한 포트폴리오 웹사이트)_

> 💡 **팁:** 처음부터 프로젝트를 정리하면 나중에 시간과 혼란을 줄일 수 있습니다!

#### 3. **Cline VS Code 확장 프로그램 설치**

VS Code 내에서 직접 Cline 확장 프로그램을 설치하여 코딩 워크플로를 향상시키십시오.

-   Cline 확장 프로그램 시작하기 튜토리얼

📺 **추천 유튜브 튜토리얼:** [<u>VS Code에 확장 프로그램 설치하는 방법</u>](https://www.youtube.com/watch?v=E7trgwZa-mk)

> ✅ **전문가 팁:** 설치 후 VS Code를 다시 로드하여 확장 프로그램이 제대로 활성화되었는지 확인하십시오.

#### 4. **필수 개발 도구**

효율적인 코딩에 필요한 기본 소프트웨어:

-   Homebrew (macOS)
-   Node.js
-   Git
