---
title: "Cline이란 무엇인가요?"
description: "VS Code의 AI 기반 개발 도우미 Cline을 소개합니다."
---

Cline은 Microsoft Visual Studio Code와 통합되는 AI 개발 도우미입니다. IDE와 LLM 간의 인터페이스를 제공하여 코드 개발을 용이하게 하고 생산성을 높이며 새로운 코더의 진입 장벽을 낮춥니다. 권한에 따라 Cline은 파일을 읽고 쓰고, 명령을 실행하고, 웹 브라우저를 사용하고, 모델 컨텍스트 프로토콜 서버를 통해 기능을 확장할 수 있습니다.

Cline을 차별화하는 것은 코드 생성에 대한 사려 깊은 접근 방식과 광범위한 통합 기능입니다. 단순히 코드 스니펫을 생성하는 대신 Cline은 솔루션을 단계별로 계획하고, 전체 개발 환경을 인식하고, 모든 변경 사항에 대한 명시적인 승인을 요구함으로써 개발자와 협력합니다. 대규모 코드베이스를 이해하고, 새로운 엔지니어의 온보딩을 가속화하고, 모델 컨텍스트 프로토콜 마켓플레이스를 통해 수백 가지 도구와 연결하여 간소화된 프로젝트 배포에서 자동화된 사고 대응에 이르기까지 모든 것을 자연어 명령을 통해 지원할 수 있습니다.
